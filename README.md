# The 7 Hills Guide — Lisbon

Interactive bilingual tour map for Lisbon city tours by tuk tuk.

**Live map:** https://ermin-muratovic.github.io/7hills

## Features
- 25+ stops with short & expandable full stories
- English / Deutsch language toggle
- 3 Hills · 5 Hills · 7 Hills route filters
- Category filters: Viewpoints, History, Churches, Hidden Gems, Culture
- Direct Google Maps links for every stop
- Mobile-optimised
